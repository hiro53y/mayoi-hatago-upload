# BGM配置ルール

このフォルダにmp3を置くと、ゲーム内で自動的に参照されます。

| 画面 | ファイル名 |
|---|---|
| タイトル | `title.mp3` |
| 1階 | `floor-01.mp3` |
| 2階 | `floor-02.mp3` |
| 3階 | `floor-03.mp3` |
| 4階 | `floor-04.mp3` |
| 5階 | `floor-05.mp3` |
| 6階 | `floor-06.mp3` |
| 7階 | `floor-07.mp3` |
| 8階 | `floor-08.mp3` |
| 9階 | `floor-09.mp3` |
| 10階 | `floor-10.mp3` |

存在しないファイルは再生されません。ブラウザの自動再生制限により、初回は画面タップやキー入力後に再生が始まる場合があります。
